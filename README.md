# Lumina Indian Bistro 🍛🤖

An AI-powered Indian fusion bistro website featuring a smart chatbot integrated with **Google Gemini AI**.

## 🚀 Features
- **Smart Chatbot**: Powered by Gemini 2.0/2.5 for natural conversation, menu exploration, and ordering.
- **Modern UI**: A premium, responsive design with glassmorphism and smooth animations.
- **Full Menu**: Categorized Indian fusion menu (Beverages, Quick Bites, Mains, Desserts).
- **Checkout System**: Integrated shopping cart with QR payment simulation.

## 🛠️ Tech Stack
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- Google Gemini API (AI Studio)

## 📦 Deployment
This project is ready for deployment on **GitHub Pages**.
