# 🛠️ Tools & Technology Stack

This project was built using a hybrid approach of **Core Web Development** and **State-of-the-art AI Tools**, demonstrating high efficiency and advanced technical implementation.

## 🤖 AI Tools Used
- **Antigravity (Agentic AI)**: Used as the primary coding partner for high-level architecture, design system implementation, and complex logic integration.
- **OpenAI GPT-4o-mini**: Integrated to power the intelligent chatbot, handling natural language processing and automated order management.
- **GitHub Copilot**: Used for real-time code autocompletion and micro-optimizations.
- **Unsplash AI**: Used for generating dynamic, context-aware imagery for the bistro menu.

## 💻 Core Technologies
- **HTML5**: Semantic structure for SEO and accessibility.
- **Vanilla CSS3**: Custom-built design system with responsive layouts, glassmorphism, and keyframe animations.
- **Vanilla JavaScript (ES6+)**: Core business logic, shopping cart state management, and asynchronous API integrations.

## 📩 Integrations
- **Formspree**: Connected to the checkout system for real-time email notifications to the bistro owner.
- **OpenAI API**: Direct integration for low-latency, high-precision chatbot responses.
